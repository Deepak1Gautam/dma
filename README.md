## 📌 About The Project

DMA (Do My Assignment) is a student marketplace platform designed to help students connect with skilled peers and access useful academic and digital services.

The platform focuses on creating opportunities for students to:

- 💼 Offer their skills and services
- 🎓 Find tutoring and mentorship
- 🛠️ Get help with academic and digital tasks
- 🌐 Build their portfolio and experience
- ⭐ Build trust through ratings and reviews

> DMA is built around learning, collaboration, and skill-based opportunities.

---

## ✨ Features

- 👤 User authentication
- 📊 Personalized student dashboard
- 💼 Student service marketplace
- 🎓 Tutoring and mentorship opportunities
- 📝 Project and service listings
- ⭐ Ratings and reviews
- 💰 Earnings and wallet system
- 📁 Portfolio showcase
- 🔍 Search and discover services
- 📱 Responsive user interface
- 🔥 Firebase-powered backend services
