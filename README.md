## 🌟 What is DMA?

**DMA — Do My Assignment** is a student-focused marketplace platform designed to connect students with skilled peers for:

* 💼 Freelancing
* 🎓 Tutoring
* 🛠️ Academic & Digital Services
* 🎨 Creative Work
* 💻 Development Projects
* 🤝 Student Collaboration

DMA creates an ecosystem where students can **learn, showcase their skills, find opportunities, and earn while growing together.**

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/Deepak1Gautam/dma.git
```

Navigate to the project:

```bash
cd dma
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```
